Package

Những gói phần mềm nhỏ chứa nhiều hàm cho các công tác đặc thù riêng.

  • dplyr: sắp xếp dữ liệu (wrangling)

  • ggplot: vẽ đồ thị dạng layer

  • stringr: thao tác với dữ liệu chữ (text)

  • …

  • install.packages(“PackageName”)

  • library(PackageName)

Nhập bảng dữ liệu vào R

File -> Import dataset -> From Excel -> …

Tìm hiểu thêm hàm read_excel

Trình bày bảng dữ liệu

Bảng dữ liệu dùng cho R nên tuân theo các nguyên tắc sau:

  • Sử dụng tiếng Anh hoặc tiếng Việt không dấu (kể cả tên file)

  • Tránh bắt đầu bằng số

    • R: bắt buộc
    • Rstudio: dùng backtick (`)

  • Tránh dùng các kí tự ngoài chữ & số (như khoảng trắng (space), !, @, …)

    • R: bắt buộc
    • Rstudio: dùng backtick (`)

  • Nếu cần cách biệt giữa hai từ, dùng . hoặc _

Sắp xếp bảng dữ liệu

nhietdo_2019 <- read_excel("../data/nhietdo_2019.xlsx")

R cho phép thực hiện nhiều thao tác trên bảng dữ liệu: chuyển hàng thành cột, gom nhiều hàng thành cột, lọc dữ liệu,…

nhietdo_2019_new <- pivot_longer(nhietdo_2019, 2:13, names_to = "Thang", values_to = "Nhiet_do")
nhietdo_2019_new$Thang <- as.numeric(nhietdo_2019_new$Thang)
filter(nhietdo_2019_new, Thang >= 6)
